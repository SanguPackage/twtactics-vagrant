TWTactics Vagrant
=================
Vagrantfile for running TWTactics on Linux or Mac.

Prerequisites
-------------
Install VirtualBox  
https://www.virtualbox.org/wiki/Downloads

Install Vagrant  
https://docs.vagrantup.com/v2/installation/

Cmd.exe
-------
`vagrant up`
